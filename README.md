Inspired by biological gene regulatory networks, GenoFlow allows users to visually construct complex, self-adapting AI architectures that combine a functional neural network layer with a stateful, trust-modulating protein network.‚ú® Key Featuresüé® Visual, Node-Based Architecture: Build and visualize your entire AI model using an intuitive, ComfyUI-like graph editor powered by LiteGraph.js.üî¨ Biologically Inspired Layers (GENREG):Protein Network (Regulatory Layer): Stateless, self-adapting "proteins" (e.g., Sensor, Trend, Comparator, TrustModifier) process environmental signals to generate a Trust Delta (fitness signal).Controller Network (Functional Layer): A simple feed-forward neural network that selects actions based on the processed signals.üå± Trust-Based Evolution: The system uses an evolutionary algorithm where Genome fitness is determined by its accumulated Trust score, promoting robust and adaptive behaviors.üêç Real-Time Environment: Train and observe your AI's behavior in a simulated Snake Environment, with real-time visualization via a separate Pygame window.üñ•Ô∏è Client-Server Architecture: A Python FastAPI/WebSocket backend handles all heavy processing (evolution, environment steps) while the JavaScript frontend provides the interactive GUI.üöÄ Quick Start1.
